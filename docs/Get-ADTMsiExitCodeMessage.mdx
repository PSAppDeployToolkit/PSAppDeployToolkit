## SYNOPSIS

Get message for MSI exit code.

## SYNTAX

```powershell
Get-ADTMsiExitCodeMessage [-MsiExitCode] <UInt32> [<CommonParameters>]
```

## DESCRIPTION

Get message for MSI exit code by reading it from msimsg.dll.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-ADTMsiExitCodeMessage -MsiExitCode 1618
```

## PARAMETERS
